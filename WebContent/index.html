＊特殊文字の表示方法

HTMLでは「<」「>」「&」「♥」を表示するのが難しく、 「<」「>」はHTMLタグと解釈されてしまうので、 これを表示させるには「&○○;」のような書き方をします。

〔例〕&lt; ⇒ <　　　
　　　&gt; ⇒ >　　　
　　　&amp; ⇒ &　　　
　　　&hearts; ⇒ ♥
　　　&quot; ⇒ "　　
　　　&#39; ⇒ '　　　


＊ページ内の移動

<h2 id="study"> のようにタグ内に id 属性で移動させたい場所の名前をつけます。
<a href="#study"> その後 a タグで このように行き先を指定します。

〔例〕
<h2 id="study" >＊　お　ま　け　＊</h2>

＊隠しリンク

一部のリンクを背景色と同色に設定し、視覚的に見えない状態にして (実際にリンクは存在してる)マウスカーソルが来たときに色が変わる仕掛けです。

スマホだとスクロールで自動的にカーソルが合うので直前のリンクとの距離を調整し タイミングを合わせると、スクロール中に突然 文字が浮かび上がることも・・

ＰＣだと上のようにマウスを重ねてもらうよう誘導しないと隠れたまま一向に発見されなかったりします

このように「隠しリンク」は あくまで発見されることを前提とした遊びのためのものです。 閲覧者の意図に反して変なサイトに飛ばしたり、閲覧者をあざむく目的では使わないようにしましょう。

ソースコードも以下に載せておきます。

〔例〕
/* HTML 本文 */
<a class="hiddenLink" href="#lovely">
　　♥ たぶん○○先生の好み？ ♥</a>

/* stylesheet/css */
// 上で指定したhiddenLinkのリンク設定
.hiddenLink{
　　color:white; //もとのリンク色を背景色と同色か似た色にする
　　text-decoration:none; //リンクの下線を消す
}

//カーソルを合わせたときのリンク設定
.hiddenLink:hover {
　　color: deeppink; //白からこの色に変わる
　　text-decoration: none;
}
